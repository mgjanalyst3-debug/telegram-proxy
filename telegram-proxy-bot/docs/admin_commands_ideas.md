# 20 ideas for admin commands

1. `/grant_30 USER_ID` — выдать 30 дней бесплатно
2. `/extend USER_ID DAYS` — продлить на N дней
3. `/reset_trial USER_ID` — сбросить пробный период
4. `/expire_sub USER_ID` — завершить подписку немедленно
5. `/reissue USER_ID` — перевыпустить логин/пароль
6. `/user USER_ID` — карточка пользователя
7. `/sub USER_ID` — текущая подписка пользователя
8. `/payments USER_ID` — история платежей пользователя
9. `/refund_last USER_ID` — вернуть последний платеж
10. `/star_balance` — баланс Stars бота
11. `/star_tx [LIMIT]` — последние транзакции Stars
12. `/ban USER_ID` — заблокировать пользователя в боте
13. `/unban USER_ID` — снять блокировку
14. `/broadcast TEXT` — массовая рассылка
15. `/broadcast_test TEXT` — тестовая рассылка себе
16. `/set_price XTR` — изменить стоимость тарифа
17. `/set_trial HOURS` — изменить длительность trial
18. `/set_limit USER_ID CONNECTIONS` — лимит одновременных подключений
19. `/note USER_ID TEXT` — внутренняя заметка по пользователю
20. `/export_payments YYYY-MM` — экспорт платежей за период
